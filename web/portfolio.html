Cloud 24 (Extreme Weather)

Cloud 24 is an app that allows users to extract pieces of information like current weather conditions, wind speed, wind direction, pressure, humidity. An extended forecast of the user's query is also provided. A user's recent search is also saved.

This project was built with HTML, CSS, and javascript.

I-Witness

I-Witness is a Team project consisting of five individuals, two backend engineers, two Ui Ux Designers, and one front end engineer, I was the front end engineer on this team.

This app allows users to report an incident report. Users can take a picture and upload a file of the incident they witness on our platform, then the I-witness team follows up from there.

This app was built using React, it has features like admin and user login/signup, image upload, and file sharing.
